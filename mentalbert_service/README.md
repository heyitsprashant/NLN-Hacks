# AI Copilot Conversation (Gemini)

## How it works

The Copilot feature uses Google Gemini to generate empathetic, context-aware conversations based on your recent journal entries. It works automatically if you have a valid Gemini API key set in backend environment variables.

## How to use

1. Write journal entries as usual.
2. Go to the Copilot section in the app.
3. Type a message or question. The Copilot will reply using Gemini, considering your recent journal context.

## How to change the Gemini API key

Set `GEMINI_API_KEY` in `backend/.env` before starting the backend.

## Notes
- Copilot does not diagnose or give medical advice.
- All conversation is generated by Gemini and is context-aware, using your last 7 journal entries and patterns.
# MentalBERT Inference Service

Local FastAPI microservice for journal classification using Hugging Face MentalBERT.

## 1) Create and activate virtual environment (Windows PowerShell)

```powershell
cd mentalbert_service
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

## 2) Install dependencies

```powershell
pip install -r requirements.txt
```

## 3) Authenticate to Hugging Face (required for gated models)

```powershell
huggingface-cli login
```

Also open and accept model terms for `mental/mental-bert-base-uncased` in your browser.

## 4) Run the API server

```powershell
uvicorn app:app --host 127.0.0.1 --port 8001 --reload
```

## 5) Test prediction

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8001/predict -ContentType "application/json" -Body '{"text":"I feel overwhelmed and exhausted today."}' | ConvertTo-Json -Depth 5
```

## Notes

- First run downloads model files from Hugging Face and caches them locally.
- Later runs load from cache and start faster.
- Service stays local and uses no paid inference API.
- `mental/mental-bert-base-uncased` is not a sequence-classification checkpoint. The service automatically falls back to `distilbert-base-uncased-finetuned-sst-2-english` for reliable journal sentiment classification.
- You can override this behavior with env vars: `MENTALBERT_MODEL_ID` and `MENTALBERT_FALLBACK_MODEL_ID`.
